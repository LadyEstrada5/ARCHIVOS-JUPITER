CREAR REPOSITORIO EN GITHUB

1. Abrir sesión en GitHub
2. En dashboard buscar la opción "new" (color verde)
3. Aparece "Create a new repository"
4. En la opción "Repository name" nombra tu repositorio (Nombre opcional: "airbnb-etl)
5. No marques la opción de "Initialize this repository with a README" si ya tienes archivos en tu carpeta local.
6. Copiar el enlace del repositorio creado. Será algo como https://github.com/usuario/airbnb-etl.git).



SUBIR EL REPOSITORIO A GITHUB MEDIANTE GITBASH

1. Abrir Gitbash

2. Si es la primera vez que usas la herramienta configúrala así: 
git config --global user.name "nombreusuario"
git config --global user.email "tucorreo@ejemplo.com"

3. Ubicar la carpeta donde está el archivo así:
find /c -iname "*nombre_de_archivo_o_carpeta*"

4. Entrar a la carpeta del proyecto con el comando cd. Por ejemplo: 
cd "/c/Users/TuUsuario/Desktop/MiProyecto"

5.Inicializar el repositorio:
git init

6. Agregar el repositorio pegando el enlace de GitHub:
git remote add origin https://github.com/usuario/airbnb-etl.git

7. Agregar los archivos que desea subir:
git add archivo.ipynb

8. Hacer el commit:
git commit -m "Primera subida del proyecto"

9. Subir el archivo a GitHub:
git push -u origin main


VERIFICAR EL REPOSITORIO EN GitHub

1. Ir a cuenta GitHub y revisar que se haya subido el archivo
